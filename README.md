## 🛠️ Installing Cisco Packet Tracer

Packet Tracer is essential for practicing CCNA labs.  
Follow the steps below to install it:

### **1. Create a Free Cisco NetAcad Account**

1.  Visit: <https://www.netacad.com>
2.  Create a free account
3.  Search for “Packet Tracer” in the course catalog
4.  Enroll in the free *“Introduction to Packet Tracer”* course
5.  Download Packet Tracer for your OS

### **2. Direct Cisco Download Page**

(Requires login)  
<https://skillsforall.com/resources/lab-downloads>

### **3. Supported Platforms**

*   Windows
*   macOS
*   Linux (.deb packages)

Once installed, you can open `.pkt` lab files in this repository.
